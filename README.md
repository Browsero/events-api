## Description

`Events REST API` is a simple API, which can be used as backend for events managing application.

## Installation

```bash
$ npm install <-- For installing all dependencies
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# For running up the database
$ docker-compose up -d
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
